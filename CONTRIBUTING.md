# Contributing

Thank you for your interest in contributing to the APM Security Scan Workshop!

## Getting Started

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request

## Workshop Content Guidelines

- Labs must follow the existing structure: frontmatter, learning objectives, exercises, verification checkpoint
- All code blocks must have language identifiers
- Screenshots go in `images/lab-XX/`
- French translations follow the same structure under `fr/`
