# Atelier — Analyse de sécurité APM

Bienvenue dans l'**Atelier d'analyse de sécurité APM** — un atelier pratique et progressif qui vous apprend à sécuriser les fichiers de configuration d'agents IA (`.agent.md`, `.instructions.md`, `.prompt.md`, `SKILL.md`, `mcp.json`) que les agents de codage IA consomment automatiquement comme instructions système de confiance.

| Labo | Sujet | Durée | Plateforme |

| [Labo 02](labs/lab-02-unicode-scanning) | Analyse de sécurité Unicode | 35 min | Agnostique |
